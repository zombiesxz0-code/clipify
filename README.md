# Clipify – AI Video Clipper

Turn any long video into 10–15 viral short clips automatically.

## Features
- AI detects the most engaging moments using local Whisper
- Auto-crops to vertical 9:16 format (Reels, Shorts, TikTok)
- Burns captions onto every clip
- Web UI works on any phone browser
- 100% free, no API key or credit card needed

## How it works
1. Upload a long video through the web UI
2. Whisper transcribes the audio for free
3. AI scores segments for viral potential
4. Best clips are cut and cropped to 9:16
5. Captions are burned in
6. Download your clips!

## Deploy on Render
1. Fork this repo
2. Go to render.com and sign up with GitHub
3. New → Web Service → connect this repo
4. Click Deploy
5. Done!
